## ▶️ Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/<your-username>/ReactJourney.git
   cd ReactJourney

   ```

2. Install dependencies:

   ```npm install

   ```

3. Run the development server:

   ```npm run dev

   ```
